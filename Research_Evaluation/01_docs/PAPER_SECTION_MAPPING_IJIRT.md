# IJIRT Paper Mapping Guide

Template used:
- `05_template/ijirt_paperformat_paper.docx`

## Suggested section mapping
- Abstract: problem, approach, and final EN/MULTI headline metrics.
- Index Terms: multilingual fact-checking, evidence retrieval, stance detection, verifier.
- I. Introduction: motivation, background, and contributions.
- II. Procedure/Method: architecture, pipeline stages, model stack.
- III. Experimental Setup: datasets, language split, runtime lock strategy.
- IV. Results: baseline vs upgraded vs final, plus confusion analyses.
- V. Discussion: error types, checkable-only interpretation, LLM impact.
- VI. Conclusion: summary, limitations, and future work.
